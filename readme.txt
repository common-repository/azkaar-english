This plugin displays a random supplication (Arabic + Translation) from Qur'an / Sunnah.

== Description ==

Azkaar English is a stand-alone plugin for daily and special-occasion supplications. The plugin is based on the popular Hisnul Muslim book by Sheikh Sa'eed Ibn 'Alee Ibn Wahf Al-Qahtaanee. Hisnul Muslim is a collection of authentic Dua and Zikr. Azkaar plugin displays a random supplication from the following topics:

* Faith (Iman)
* Ablution (Wudu)
* Prayer (Salah)
* Fasting (Sawm)
* Pilgrimage (Hajj, Umrah)
* Sacrifice (Qurban)
* Seeking Allah's Refuge
* Forgiveness, Repentance
* Remembrance of Allah
* Seeking Guidance (Istikharah)
* Difficulty, Fear, Distress
* Happiness, Amazement
* Everyday: Morning, Evening
* Mosque
* Manners, Ethics
* Dealings, Affairs
* Home
* Family
* Clothes, Garments
* Guest
* Food
* Toilet
* Health
* Wealth
* Sleep
* Weather
* Travel
* Hereafter, Death

== Installation ==

1. Upload azkaar-english plugin folder to `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add "Azkaar English" widget through 'Appearance > Widget' menu to your wordpress theme's sidebar
